Computer Network Topologies

Star Topology

A Star Topology is a network configuration where all devices connect to a central hub, such as a switch or router.

Advantages:
• Easy to install and manage
• Failure of one node does not affect others
• High performance due to dedicated connections

Disadvantages:
• Entire network depends on the hub
• More cabling required
• If hub fails, whole network goes down

Ring Topology

A Ring Topology connects devices in a circular loop, where each device is connected to exactly two others.

Advantages:
• Reduces data collisions
• Easy to install and expand
• Predictable data flow

Disadvantages:
• Failure of one node breaks the entire ring
• Troubleshooting is difficult
• Performance decreases as network grows

Bus Topology

A Bus Topology connects all devices to a single communication cable known as the backbone.

Advantages:
• Easy and inexpensive to install
• Requires less cable
• Good for small networks

Disadvantages:
• Backbone failure stops the entire network
• Slow with heavy traffic
• Troubleshooting cable faults is difficult

Tree Topology

A Tree Topology is a hierarchical structure combining characteristics of both star and bus topologies.

Advantages:
• Highly scalable
• Easy to add new nodes
• Fault isolation is simple

Disadvantages:
• Backbone failure affects entire network
• Expensive cabling
• Complex to design

Mesh Topology

A Mesh Topology connects each device to multiple others, offering multiple data paths.

Advantages:
• Very reliable (redundant paths)
• High security
• Fast data transmission

Disadvantages:
• Very expensive to install
• Requires lots of cabling
• Complex setup and maintenance
